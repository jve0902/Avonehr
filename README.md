# clinios

## Stack

- React.js
- Redux
- Material-ui
- Node + Express
- PostgreSQL

## Client

Setup:

1. `cd client`
2. `npm install` or `yarn`
3. copy `.env.sample` into `.env` and fill in credentials

To run:

1. `yarn start` or `npm run dev`
2. Open [http://localhost:3000](http://localhost:3000)
